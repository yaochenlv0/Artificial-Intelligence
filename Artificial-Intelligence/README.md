# Artificial-Intelligence

这是我的人工智能学习记录仓库，主要用于整理在学习大模型应用开发过程中的代码实践、知识总结、阶段复盘与基础补充内容。

目前仓库内容以“边学边做”的方式推进，围绕最基础的大模型调用、简单问答程序、文本总结助手等内容展开，逐步积累对人工智能应用开发的理解，同时沉淀学习过程中的代码、笔记与问题记录。

---

## 仓库内容

本仓库当前主要包含以下几部分：

### 1. Day 学习记录
以 Day1、Day2、Day3 的形式记录每天的学习主题，通常包含：

- 学习代码
- 核心知识点理解
- 当日总结
- 常见问题记录

### 2. Python 基础补充
用于整理人工智能学习过程中涉及到的 Python 基础知识，包括：

- API 基础
- 文件读取与处理
- GitHub 命名规范
- 成长路线规划

### 3. 学习过程沉淀
不仅记录“做了什么”，也记录：

- 为什么这样做
- 代码各部分的作用
- 遇到了哪些问题
- 如何理解和优化
