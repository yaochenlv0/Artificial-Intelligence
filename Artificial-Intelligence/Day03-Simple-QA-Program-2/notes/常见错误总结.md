# 常见错误

| 错误类型              | 错误场景         |
| ----------------- | ------------ |
| FileNotFoundError | 文件不存在        |
| KeyError          | 字典里没有这个键     |
| IndexError        | 列表索引超出范围     |
| ValueError        | 值类型正确但类型不对   |
| TypeError         | 类型不匹配        |
| ZeroDivisionError | 除以0          |
| AttributeError    | 对象没有这个属性或者方法 |
# 文件操作有关

| 错误类型                                 | 错误场景             |
| ------------------------------------ | ---------------- |
| ConnectionError                      | 网络连接错误           |
| TimeoutError                         | 请求超时             |
| requests.exceptions.RequestException | requests库的各种网络错误 |
# 数学相关

| 错误类型            | 错误场景     |
| --------------- | -------- |
| OverflowError   | 数字太大超出范围 |
| ArithmeticError | 算术错误（父类） |
